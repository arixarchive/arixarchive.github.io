Change Log

Arix Archive v1.4, build 1546 (28.06.2024)
28th June 2024
  • + Security Updates.
  • + Fixed Crashed when opening Benchmark.
Arix Archive v1.3, build 1540 (24.06.2024)
24th June 2024
  • + Fixed ValorPE error when archiving PDF files.
  • + Added an alternative function to prevent errors when generating unique Vectors and Salt during archive encryption.
  • + Ends support for Windows 2000/ME/XP. but Vista and 7 can still be run.
Arix Archive v1.2, build 1535 (14.02.2023)
14th February 2023
  • + Optimize the encryption and decryption process by adding the WEBEC_aCsl virtual system.
  • + Change the layout of the UI and make important buttons clearer.
  • + Fixed error/crash when running on Windows 11.
  • + Fixed the bluescreen issue.
Arix Archive v1.1, build 1310 (22.09.2017)
22nd September 2017
  • + Initialize the vector with the HWID and id of the processor.
  • + Set the dictionary size to 64MB (from 24MB previously).
Arix Archive v1.0, build 1020 (12.05.2017)
12th May 2017
  • + Fixed conflicts when extracting files over 100MB in size.
  • + Fixed errors when reading and archiving audio files (mp3 and wav).
  • + Fixed the function to delete temporary files.
  • + Change the structure of dictionaries and datasets to make compressed files easier to initialize.
  • + Added Password Manager feature.
  • + Change the UI to be more flexible.
Arix Archive Beta, build 0001 (29.08.2015)
29th August 2015
  • + First Release.
